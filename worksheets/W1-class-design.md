## 1. โจทย์ของทีม

**Problem statement** (1-2 ประโยค บอกว่าจะแก้ปัญหาอะไรในชีวิตจริง):
เกิดปัญหาการขโมยของในห้องเรียน และไม่สามารถระบุคนขโมยได้อย่างแน่นอน
```
[โครงงานนี้เป็นการพัฒนาระบบรักษาความปลอดภัย (Safety System) สำหรับเฝ้าระวังทรัพย์สินมีค่าในช่วงเวลาที่กำหนด เช่น ช่วงพักกลางวัน โดยใช้อุปกรณ์ติดตั้งบริเวณหน้าห้องเรียน ระบบทำงานผ่านบอร์ด Arduino UNO Q ร่วมกับอุปกรณ์ต่าง ๆ ได้แก่ Modulino Distance สำหรับตรวจวัดระยะห่างของบุคคลหรือวัตถุที่เข้าใกล้พื้นที่เฝ้าระวัง, Modulino Thermo สำหรับตรวจวัดอุณหภูมิ, Modulino Buttons สำหรับส่งสัญญาณแจ้งเตือนฉุกเฉิน (SOS), Modulino Buzzer สำหรับส่งเสียงแจ้งเตือน, และ Modulino Knob สำหรับปรับระดับความไวของระบบตรวจจับ

นอกจากนี้ ระบบยังเชื่อมต่อกับกล้องเว็บแคมเพื่อบันทึกภาพเมื่อมีการตรวจพบความผิดปกติ พร้อมทั้งส่งข้อมูลแจ้งเตือนไปยังเว็บไซต์แดชบอร์ด เพื่อให้สามารถติดตามและตรวจสอบสถานการณ์ได้แบบเรียลไทม์]
ตัวอย่าง: เราจะสร้างระบบตรวจจับการล้มของผู้สูงอายุ
โดยใช้ Modulino Movement ติดที่เอว เพื่อช่วยแจ้งเตือนเมื่อเกิดเหตุ
```

## 2. Class Definition

ระบุ class ที่จะใช้ + เกณฑ์ที่ทำให้แต่ละ class แตกต่าง:

| # | ชื่อ Class | คำอธิบาย (3-4 ประโยค) | สิ่งที่ทำให้ต่างจาก class อื่น |
|---|---|---|---|
| 1 | อุณหภูมิสูง | เมื่อมีคนเข้าใกล้อุณหภูมิจะสูงกว่าอากาสปกติจะแจ้งเตือนทันที ติดหน้าห้อง | temp มากกว่า 36.4|
| 2 | มีอะไรมาใกล้ | เมื่อมีของหรืออะไรก็ตามที่ขยับมาใกล้จะแจ้งเตือนทันที ติดหน้าห้อง| ใกล้มากกว่า30 ซม. |
| 3 | สัญญาณSOS | กดปุ่มเมื่อคิดว่าอาจจะมีคนเข้าไปแล้ว 2 เซนเซอร์ไม่ทำงาน ค้างไว้ 3 วินาทีจะแจ้งเตือนทันที เอาไว้กับครู| กดปุ่มค้าง button 3 วินาที|

**ตัวอย่าง (Track A, Fall Detector):**

| # | ชื่อ Class | คำอธิบาย | สิ่งที่ทำให้ต่าง |
|---|---|---|---|
| 1 | ยืน | ผู้ใช้ยืนนิ่ง หรือยืนเฉยๆ ใส่ที่เอว | accel แกน Z สูง, gyro ต่ำ |
| 2 | เดิน | เดินปกติ ก้าวเท้าสลับ | accel oscillation, periodic |
| 3 | ล้ม | ล้มลงพื้นกะทันหัน | high accel spike + horizontal Z |

## 5. Diversity Plan — กันอคติ (Bias)

ตอบ 3 คำถาม:


### a) มีหลาย "environment" ไหม?
```
- ห้องเรียนแสงปกติ (กลางวัน)
- ห้องเรียนแสงน้อย (ไฟปิด)
- มุมกล้องต่างกัน (ตรง / เฉียง)
[เก็บในห้องเรียน + ห้องอื่น? แสงสว่าง/แสงน้อย? เวลาต่างกัน?]
```

### b) มีหลาย "subject" ไหม? (กรณี Vision/Audio/Motion)
```
- สมาชิกทีม 3 คนเดินผ่านกล้อง
- ความสูงต่างกัน
- เสื้อผ้าสีต่างกัน
[เก็บจากหลายคนในทีม? ลักษณะ background ต่างกัน?]
```

### c) มีหลาย "variation" ไหม?
```
- ระยะห่างจากกล้อง 1-3 เมตร
- ความเร็วในการเดินช้า/เร็ว
- เดินตรง / เดินเฉียง
[มุม/ระยะ/ความเร็ว/ระดับเสียง ที่แตกต่างกัน?]
```
